# 🎓 Student Enrollment System (Angular + Node.js + MySQL + JWT)

A full-stack web application for managing student records with secure authentication and smooth frontend-backend integration.

---

## ✨ Features
- 🔐 **JWT Authentication** – Secure login & signup
- 👨‍🎓 **Student Management** – Registered bulk students in a course, marks assign to students, view students data, login/logout
- 📡 **REST API Integration** – Clean separation between frontend & backend
- 📱 **Responsive UI** – Built with Angular
- 💾 **MySQL Database** – Relational schema for structured data

---

## 🧰 Tech Stack
- **Frontend:** Angular, HTML5, CSS3, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Auth:** JWT
- **Learning (Next Steps):** Distributed Systems, Cloud-Native Tools (Kubernetes, Apache Iceberg), System Design
